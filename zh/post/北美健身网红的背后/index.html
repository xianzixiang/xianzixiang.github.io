北美健身网红的背后

最近更新于 Mar 14, 2020 7 分钟阅读时长

知乎有一个这个话题

https://www.zhihu.com/question/337568352

为什么好多黑人健身后肌肉很强壮，而在中国却不常见这种肌肉强壮的人呢？

当时看了几个答案，我觉得其他人已经解释得差不多了，没什么需要补充，直到我看见赞数最高的答案的时候。

答案里就放了这位健美选手的照片，没什么特别。

看了评论以后就提醒了我，这反映了一个很常见的现象，值得讨论一下。

首先这位健美选手并不是随便一位黑人模特，他的名字是Ulisses Jr，四次Musclemania冠军，在外国的健美世界可以说是挺有名的。

大家应该认识一下他。

男性乳房发育症在健美世界里是很普遍的一个后卫症，其中一个主要原因是类固醇的副作用。当你做一个类固醇周期的时候，自身的睾酮分泌会下降，同时身体可以把类固醇转换成雌激素和催乳激素。缺乏适量的雌激素阻滞剂的时候，乳房就增大。这个治疗类固醇副作用和恢复自身激素分泌的结后周期叫PCT（Post Cycle Therapy）。

这当然是一个非常简化的说法，至于Ulisses Jr的案件，估计是他的PCT系统出了问题吧。

这是Ulisses Jr和他同伙Simeon Panda。Ulisses连乳房都长了，而Simeon比Ulisses还要大，他还坚持自称一辈子从来没有碰过类固醇，你觉得呢？

简单说，整年保持这个水平的体脂和肌肉，天然的话是不可能的，除非你用克仑特罗或者DNP。

人家用什么药是他们的事儿，我不在乎，我在乎的是中国人对这方面的误解。

中国的健身文化特别不成熟，普通人缺乏很多这方面的基础学问，结果就很无妨、很天真，把许多许多根本不可能的形象好像视为外国人基因优势的证据。

网络上四处都会有此类型的照片，有模特、演员、运动员，很多影像都满身明显是人工的肌肉，可是当国人看见他们的时候，第一个反应是哇，外国人厉害，而不是他们跑的类固醇堆里用的群勃龙有多少。

这就是我在之前这个答案的评论区里发现的，没有一个人有这个概念，没有一个人发现这些模特明显是不正常的，没有一个人直接点出这个问题。

北美健美运动员Dallas McCarver的验尸报告

正常男性的睾酮水平是大概300-1000 ng/dL；他的睾酮水平是550 ng/mL（55000 ng/dL）。

正常的睾酮比例（T/EpiT）是4以下；他的比例是130。

我还需要说什么？

当一个人的睾酮水平是正常人的100倍以上，血管里的睾酮是99%人工的，你觉得，到底天赋是他长肌肉的能力，还是他对药物的反应和副作用的抵抗力呢？

“我有用（类固醇），一直到比赛的时候，八至九个星期之前，这是每个人都需要用的一个东西，如果你想有机会的话。”

黄金时期的阿诺德施瓦辛格（包括他健美舞台上面对的所有对手）基本上是半人半类固醇，阿诺德本人也承认过用大力补和其他类固醇。

据阿诺德的健美同伙Rick Wayne说，阿诺德从13岁就开始用类固醇。

真相，谁知道呢。

假如年轻的阿诺德是纯天然的话，那他的天赋就确实是很好的。

假如没有类固醇的存在，理论上他应该一样会很有成就。

Chris Bumstead：2019 Mr Olympia Class Physique #1

2019届的奥林匹克先生刚刚结束了，加拿大选手Chris Bumstead就获今年的Classic Physique 冠军。

年轻的他没什么特别，好像没有反映到今天的冠军身材。

Alina Popa：奥林匹克小姐亚军

你觉得呢？

这儿的意思当然不是说类固醇定夺一切了，可是你自己可以分析一下。

还有另外一个感觉，很多人好像觉得如果一个人不是奥林匹克先生水准的话，他就是天然。

这也是非常天真的。

三名北美网红

以上的，你觉得谁有用类固醇呢？

普通人会觉得他们三个都是天然的，对这个方面有熟识的人会觉得左边或者中间的用类固醇，主要是中间的，不过事实是什么？

他们三个都有用类固醇（左：Test、HGH、Epistane。中：基本上是有两条腿的药库，数不尽。右：Test、Tren、Winstrol）。

其实这三个身体大致上都可以纯天然地练成，除了部分问题（左边的腹部好像有点儿HGH引起的膨胀。中间的，类固醇引致的胸肌和三头肌不符合他的身体比例，整年保持这个水平的体脂也是不可能的），不过在现代社会，尤其是西方世界，谁真的肯什么都不用，花五六年去锻炼自己呢？谁有这股耐性呢？特别是当你有研究显示，如果你用类固醇然后什么东西都不做，你长的肌肉会比你去做天然健身的多（https://www.nejm.org/doi/full/10.1056/NEJM199607043350101）。

15岁的“天然”健美：更可怜的是竟然真的有人会相信

在外国，类固醇可以很便宜，也非常普及，很多年轻人在高校学生的年纪的时候就会开始打类固醇针，为了运动效能或者想自己好看一点儿，神不知鬼不觉，有多少人真的可以抵抗到这个引诱？

本来健身就已经要求很大的付出：

  1. 伙食：给你提供足够微量营养素和宏量营养素的健康饮食，每天吃3000至4000卡路里（然后旺季需要燃脂的时候就减低脂肪或者碳水化合物，保持高蛋白）
  2. 休息：充足的睡眠和休息
  3. 锻炼：渐进超负荷力量训练，一个星期去练五天，每一天针对一两个身体部位，每一组保持在6-12 Rep的范围内
  4. 精神：一股充满上进心的精神，永不放弃，一直保持你的健身计划

当你一至四都执行得完美，一贯在健身房练三四年，实现了你九成以上的个人潜能，以后你很难再会有突破的。

剩下的只有第五。

可以参考：https://www.bilibili.com/video/av40842107/

肌肉还不够大的话，打一点儿Test。

腹肌不够显现，用一点儿Clen。

有种的话，还可以尝试一下Synthol。

挺可惜的，因为：

第一：变质

论形态美感，除非你是受了外国的畸形肌肉文化影响，天然总是最舒服的。

Rich Piana死后，家里被发现有20瓶类固醇，他的粉丝大部分是年轻男士

第二：病态

这个趋势引起体象障碍，带领了许多人，特别是天真的年轻人，去追求一个根本就是病态的外形。无数多的年轻人被误导，牺牲了自己的健康和外表为了肌肉发达。

非天然的健美模特告诉你应该继续努力，买补剂

第三：骗局

大部分模特不是有一个明显是非天然的身材，就是表面看起来天然，背后就暗中用禁药，半年时间练成需要三年的身体，然后从中就哗众取宠，给普通人推销一大群完全没用的补剂和训练计划。

Jesse Ventura：“80年代的时候，类固醇是合法的，当时的摔跤手用类固醇如吃糖果一样。”

第四：假象

为什么白人的肌肉特别大？为什么黑人的肌肉特别大？

无论是模特、运动员、还是演员什么的，他们大部分和类固醇是分不开的，特别是以貌取财的人士，满身肌肉的身材是很多外国模特和演员的饭碗。

UFC拳手Alistair Overeem说吃马肉就可以长肌肉，结果他的睾酮比例被验出是14比1

第五：无奈

在很多事业上，唯一存在的就只有竞争。

别人用的话，你怎么呢？

你500ng的天然睾酮对人家5000ng的人工睾酮，你有把握吗？

你会坚持自己的原则，石投大海，还是跟风，然后毁灭你的对手呢？

Eugen Sandow（1867-1925）：现代健美之父

合成代谢类固醇是发明于1930-1940并且流入民间，这个时候Eugen Sandow已经过世了，那我们可以99.9%肯定他的身材是天然健身和超凡基因的产物。

这当然不是说没有人能够超越他。潜力更强的人，虽然罕见，不过依然会有的，哪个民族都会有。只不过，在这个年代，有了类固醇的发明以后，突然间随便一个国外的普通网红或者模特，甚至连一些青少年或者女性的健美运动员，都比Eugen Sandow巨大。

对，现代社会有一些所谓新的练习技巧，可是增肌，归根结底依然是围绕着渐进式超负荷，没什么特别。

至于这些健美界的补剂，就如我刚才说的，事实就是九成以上的补剂都是骗局。会进入你身体的，真正有效的只有食物营养和类固醇。

如果单靠“天赋”这个词语去解释的话，这就真的有点儿不诚实了。

我希望的，就是更多国人可以明白这些道理。

别任何影像都接受，也别为自己立下一些完全不现实的目标。

我觉得，最好自己体验一下。

跟随这四点：

  1. 吃一个给你提供足够微量营养素和宏量营养素的健康食谱
  2. 充足的睡眠和休息
  3. 渐进超负荷力量训练
  4. 保持一股充满上进心的精神

去健身三年，之后我保证你会明白。

说不定，可能你有所谓超凡脱族的天赋呢。
